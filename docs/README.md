# Documentation

This directory contains the product and engineering notes for OpenClaw Mission Control.

## Guides

- [Object Model](object-model.md): core product entities and relationships.
- [Local Gateway](local-gateway.md): staged plan for native desktop and local worker capabilities.
- [Security Model](security-model.md): approval classes, blocked actions, and trust boundaries.
- [Deployment](deployment.md): local development, static distribution, and future desktop packaging.
- [Vercel Deployment](vercel.md): Vercel build settings, deployment routes, and optional platform capabilities.
- [API Roadmap](api-roadmap.md): future command, event, and integration contracts.

The current application is a local-first browser prototype. The documents describe the intended path toward a safer desktop product with explicit approvals and auditable local automation.
